# FA templates (backend)

This folder holds the shared Factory Audit Word template used by every user
of the portal. Do **not** rely on a Desktop copy per machine.

| File | Purpose |
|------|---------|
| `FA_template.docx` | TÜV Rheinland FA report template filled by `fa_automation.py` |

Replace `FA_template.docx` here when the official template is updated.
All portal deployments resolve this path automatically.
